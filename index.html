1. Data Collection

We do not collect any personal information without user consent. The bot does not store or transmit personal data to third-party servers. However, the following data may be collected automatically during bot usage:

  • Discord username.
  • Server and channel ID where the bot is used.
  • Messages sent by the user to the bot for task execution.

2. Use of Collected Information

The data collected during bot usage is only used to ensure the functionality of the services provided by the bot, such as command execution and improving the bot’s performance.

3. Data Retention

We do not retain user data on a permanent basis. All data is automatically deleted after command execution or session termination.

4. Data Protection

We take the necessary technical measures to protect the collected information from unauthorized access.

5. Changes to Privacy Policy

We reserve the right to change this privacy policy at any time. Users will be notified of changes through updates in the bot's description.

6. Contact

If you have any questions about this privacy policy, you can contact us via [Contact Information].

1. Datenerhebung

Wir erheben keine persönlichen Informationen ohne die Zustimmung des Benutzers. Der Bot speichert oder überträgt keine persönlichen Daten an Dritte. Folgende Daten können jedoch automatisch während der Nutzung des Bots gesammelt werden:

  • Server- und Kanal-ID, in denen der Bot verwendet wird.
  • Nachrichten, die vom Benutzer an den Bot gesendet werden, um Aufgaben auszuführen.

2. Verwendung der gesammelten Informationen

3. Datenspeicherung

Wir speichern Benutzerdaten nicht dauerhaft. Alle Daten werden automatisch nach der Befehlsausführung oder dem Sitzungsende gelöscht.

4. Datenschutz

Wir ergreifen die notwendigen technischen Maßnahmen, um die gesammelten Informationen vor unbefugtem Zugriff zu schützen.

Wir behalten uns das Recht vor, diese Datenschutzrichtlinie jederzeit zu ändern. Benutzer werden über Änderungen durch Updates in der Beschreibung des Bots benachrichtigt.
